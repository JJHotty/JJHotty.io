Reflection on Branch, Pull Merge.
What is Main?
Main is the root directory of a GitHub repo. It’s the main line and branches are made of of it. It’s also what people will clone to their devices to make local.

Why create a branch?
Branches allow us to keep the main line just well developed code without issues. And we can troubleshoot and work on the code in branches meaning the main line is just the working project. It also allows multiple people to work on various parts of the app without getting in each others way by moving or deleting code without it affecting the others work.
